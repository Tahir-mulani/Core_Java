## üõ†Ô∏è Prerequisites

To run the Java programs in this repository, you need:

- **Java Development Kit (JDK)**: Version 8 or higher
- **IDE**: Any Java IDE like Eclipse, IntelliJ IDEA, or VS Code with Java extensions
- **Git**: For cloning the repository

### Installation

1. Download and install JDK from [Oracle's website](https://www.oracle.com/java/technologies/javase-downloads.html) or use OpenJDK.
2. Verify installation:
   ```bash
   java -version
   javac -version
   ```
